# Flippin website (www.flippin.app)

Flippin is an iOS/iPadOS flashcard app for language learning. This domain hosts static marketing pages only.

## Primary actions for users

- Download the app from the App Store listing linked from the homepage.
- Contact support at support@flippin.app or the Support page.

## API surface

There is no public REST API for user flashcard data on this hostname. Discovery metadata is published for automation (see `/.well-known/api-catalog`).
